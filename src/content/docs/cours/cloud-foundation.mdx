## Points clés du cours

- **Fondamentaux cloud**  Définition NIST, 5 caractéristiques essentielles, IaaS/PaaS/SaaS, modèles de déploiement
- **AWS CAF**  6 perspectives (Business, Personnes, Gouvernance, Plateforme, Sécurité, Opérations)
- **Infrastructure globale**  39 régions, zones de disponibilité, console, CLI, SDK
- **EC2**  Types d'instances, modèles d'achat (On-Demand, Reserved, Spot, Savings Plans)
- **Haute disponibilité**  ELB, Auto Scaling Groups, scalabilité verticale vs horizontale
- **Sécurité**  Responsabilité partagée, IAM, MFA, KMS, Secrets Manager, GuardDuty, WAF
- **Bases de données**  RDS, Aurora, DynamoDB, ElastiCache, Redshift, DocumentDB, Neptune
- **Serverless**  Lambda, API Gateway, SQS, SNS, EventBridge, Step Functions
- **Stockage**  S3, EBS, EFS, classes de stockage, Snow Family
- **IaC**  CloudFormation, Terraform
- **Gouvernance**  AWS Organizations, SCP, Control Tower, CloudTrail, Config
- **Certification**  CLF-C02 : structure, domaines, conseils
